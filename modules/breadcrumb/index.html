    Source API
    Breadcrumbs are used to show navigational hierarchies between contents. You can easily create breadcrumbs to show users their location within a website. To apply this component, add a container element with the .breadcrumb class and add child elements with the .item class.
    Punica Modules Breadcrumb
    Sizes of Breadcrumb

    Add the .small and .large class to a breadcrumb to make it smaller or larger.
