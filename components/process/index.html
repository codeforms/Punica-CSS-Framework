    Source API
    The Process component is used to visually represent a sequence of steps or stages in a process or workflow. It helps users easily follow the progression of tasks or actions, often with the use of numbered steps, icons, or progress indicators. The Process component can be customized to fit various design styles and is perfect for guiding users through multi-step tasks or processes. Add a container element with the .process class and add child elements with the .item class. Also through the .active or .failed class will be highlighted items and indicate the current state of process.
