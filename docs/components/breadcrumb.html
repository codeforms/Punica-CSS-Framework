Breadcrumbs are used to show navigational hierarchies between contents. To apply this component, add a container element with the .breadcrumb class and add child elements with the .item class.

Props Default Class Name
Container .breadcrumb
Breadcrumb Item .item
Active Item .active
Sizes Default Class Name
Small .small
Large .large

</div>

Sizes

Add the .small and .large class to a breadcrumb to make it smaller or larger.
