Card Groups & Equal Height Cards

You can choose an arbitrary card count per row. Default system allows maximum 16 cards each row. The example below shows three cards (.cards .three) will fit in the first row and the fourth card dynamically moves to the second row. Also card groups automatically uses flex styles to match height between cards in the same row.
