Panels can be used to separate the content of a page. To apply this component, just add the .panel class to a container element.

Panel Title
launch person
Lorem ipsum dolor sit amet, eu duo inermis vocibus.

Eam insolens elaboraret in, sumo habeo nec ne. Dicta animal integre est cu, sea tacimates percipitur.
