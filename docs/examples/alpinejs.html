Punica Examples Alpine.js
build_circle Alpine.js error_outline Optional

You can easily integrate the Punica with the Alpine.js for javascript interactivities. Alpine.js is a rugged, minimal frontend development framework for adding JavaScript behavior to HTML markups.

If you want to add Alpine.js to your project, add the following script tag to the section of your layout file.

HTML Markup
<!DOCTYPE html>
<html>
    <head>
        <title>Title</title>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/codeforms/Punica-CSS-Framework@1.7.5/dist/punica.min.css" crossorigin="anonymous">
        <script src="https://cdn.jsdelivr.net/gh/alpinejs/alpine@v2.x.x/dist/alpine.min.js" defer></script>
    </head>
    <body>
    </body>
</html>
