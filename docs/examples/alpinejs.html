toys Alpine.js Integration error_outline Optional

You can easily integrate the Punica with the Alpine.js for javascript interactivities. Alpine.js is a rugged, minimal frontend development framework for adding JavaScript behavior to HTML markups.

If you want to add Alpine.js to your project, add the following script tag to the section of your layout file.

HTML Markup
<!DOCTYPE html>
<html>
    <head>
        <title>Title</title>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/codeforms/Punica-CSS-Framework@1.5.20/dist/punica.min.css" crossorigin="anonymous">
        <script src="https://cdn.jsdelivr.net/gh/alpinejs/alpine@v2.x.x/dist/alpine.min.js" defer></script>
    </head>
    <body>
    </body>
</html>
Tab

Here's an example of using AlpineJS for "tab" functionality inside a Punica component's view.

Tab 1 Tab 2
Tab 1 Content
Tab 2 Content
<div x-data="{tab: 1}">
    <div class="tab">
        <a :class="{'active': tab == 1}" @click.prevent="tab = 1" href="#" class="button">Tab 1</a>
        <a :class="{'active': tab == 2}" @click.prevent="tab = 2;" href="#" class="button">Tab 2</a>
    </div>
    <div class="panel pure reset full width">
        <div class="content">
            <div x-show="tab == 1">
                Tab 1 Content
            </div>
            <div x-show="tab == 2">
                Tab 2 Content
            </div>
        </div>
    </div>
</div>
