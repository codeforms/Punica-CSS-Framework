### Usage
You can copy-paste the theme array (for example; palenight) to ```_themes/index.scss```;
```scss
    // _themes/index.scss
	$punica-themes: (
		palenight: (...)
	)
```
..and update ```$active-theme``` variable in ```punica.scss```; 
```scss
    // punica.scss
    $active-theme: palenight;
```
Then compile the Punica again and ready to engage!
